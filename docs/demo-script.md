# Demo Script

1. Open the product home page and say that it solves the queue problem by letting users check the menu before going to the cafe.
2. Show the three main sections: Food, Drinks, Sushi and Rolls.
3. Open one category and explain that the menu comes from the backend and PostgreSQL, so staff can update it without changing frontend code.
4. Switch to English and show the same flow for international users.
5. Use search to find one item quickly.
6. Open Admin Panel and log in as owner.
7. Show the team list with owner and workers.
8. Edit or add one menu item and explain that workers can update the menu, while the owner can also manage staff.
9. End on the deployed URL and GitHub repo.
